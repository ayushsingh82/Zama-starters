# zama-starters

A collection of starter dapps and live demos built on the Zama Protocol (fhEVM), showcasing privacy-preserving computation and encrypted data processing.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Pages

- `/` — landing page
- `/explore` — live Zama dapps built by the community
- `/swap` — confidential swap demo UI
